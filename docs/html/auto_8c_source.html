108 /*
109  * Runs the user autonomous code. This function will be started in its own task
110  * with the default priority and stack size whenever the robot is enabled via
111  * the Field Management System or the VEX Competition Switch in the autonomous
112  * mode. If the robot is disabled or communications is lost, the autonomous
113  * task will be stopped by the kernel. Re-enabling the robot will restart the
114  * task, not re-start it from where it left off.
115  *
116  * Code running in the autonomous task cannot access information from the VEX
117  * Joystick. However, the autonomous function can be invoked from another task
118  * if a VEX Competition Switch is not available, and it can access joystick
119  * information if called in this way.
120  *
121  * The autonomous task may exit, unlike operatorControl() which should never
122  * exit. If it does so, the robot will await a switch to another mode or
123  * disable/enable cycle.
124  */
